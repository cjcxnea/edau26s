# 页面迁移通知改造 — 工作概览

## 完成内容

将 `index.html`（原四六级成绩查询页）的全部内容删除，重新设计为**站点迁移通知页**：

- **停止更新提示**：醒目展示"本页面即将停止更新"，说明页面已停止维护
- **跳转按钮**：主按钮"前往新页面"，指向 https://cjcxnea.github.io/educet/
- **新网址展示**：网址以等宽字体芯片样式展示，附"复制"按钮（带复制成功反馈，含降级方案），提醒用户收藏保存

## 设计要点

- **简洁现代**：单卡片居中布局，单一蓝色主色（#2563eb）+ 中性灰文字，无多余装饰
- **信息直观**：图标 → 标题 → 说明 → 主操作 → 网址保存，视觉层级清晰
- **暗色适配**：通过 `prefers-color-scheme` 自动适配深浅色
- **无障碍**：焦点可见样式、44px+ 触控目标、`prefers-reduced-motion` 支持、语义化标签
- **自包含**：样式与脚本全部内联，不再依赖 css/ 与 js/ 目录资源

## 备注

- 原有 `css/`、`js/`、`img/` 等资源文件未删除（`score.html` 可能仍在引用），如需清理可另行处理
